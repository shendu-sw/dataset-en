任务

构建本数据集是为评价热源系统温度场重建方法提供参考。本数据集中，热源系统区域为$0.1m \times 0.1m$的矩形。该矩形区域划分为$200 \times 200$​​​的网格进行离散化表示，且组件也采用离散化形式放置在网格中。

（1）边界条件。根据实际工程要求和任务的通用性，选取散热器和正弦波分布边界作为具有代表性的边界条件

散热器是一种被动热交换器，它将电子或机械设备产生的热量转移到流体介质（例如空气或液体冷却剂），设备产生的热量从散热器出消散，从而允许调节设备的温度。散热器广泛应用于电子领域，已成为现代微电子学的重要组成部分。下图展示了典型的散热器，散热器上的温度恒定，为$T_0$。

为了提高数据集的通用性，考虑了带有温度分布的边界条件。分布可以是恒温的，也可以是更复杂的，本数据集采用正弦波分布作为代表。这种边界条件被称为正弦函数边界条件，是热分析研究中另一种典型的狄利克雷边界条件。边界上的温度值可以表示为：

综上，本数据集考虑三个具有代表性的案例：HSink、ADlet 和 DSine。

  • 案例 1：HSink。带有散热片的热源系统，散热器$\delta$​ 的宽度设置为 0.01m，恒温值$T_0$为 298K （Dirichlet 边界），其他边界是绝热的（Neumann 边界）。内部热源采用 A 型配置。
  • 案例 2：ADlet。具有不同 Dirichlet 边界条件的热源系统，其中一个边界上温度设为正波分布，其他边界为恒温值$T_0$​​=298K。内部热源采用 B 型配置。
  • 案例 3：DSine。具有一个正弦波分布边界的热源系统。其他三个边界都是绝热的（NeumannBC），内部热源采用 C 型配置。

系统中每个热源的功率强度（或高斯功率分布下的最大功率强度）为$0 \sim 30000W/{m^2}$​。

我们将任务分为以下四种：

（1）Point-based Methods。基于点的温度场重建方法以某热源系统为例进行研究，这些方法试图学习从某一点坐标到相应温度值的映射函数。对于每个实例，将系统域划分为PoIs和监测点，其中监测点用作训练样本，PoIs用作测试样本。

（2）Vector-based Methods。尽管基于点的方法很容易实现，但它只执行一个实例一个任务，而另一个实例必须解决其他实例的优化问题。这将大大增加重建过程的成本时间。为此，Vector-based目标是学习一类样本的温度场重建，这类方法学习监测点的温度向量与PoIs的温度向量之间的映射关系。

（3）Image-based Methods。基于矢量的方法是实现一类任务最简单的方法，通常忽略了点和监测点之间的物理和空间相关性。通过域离散化，可以将温度场重建任务计算为一个图像到图像的回归问题。它学习了监测点的温度矩阵与域整体温度场的映射关系。一般采用基于图像的深度回归模型。

（4）Graph-based Methods。基于图的方法是利用系统的物理和空间相关性的另一种方法，利用图来描述监测点与PoIs之间的关系。

测试数据说明

随着功率强度的变化，随机生成不同的样本。理论上，可以产生任何可行功率强度的热源系统。然而，仍有较低的概率产生某些特定的样本，如一个或多个零功率强度的样本。为了充分验证重构性能，数据集提出几种特殊规则生成一些特殊的样本，形成了多样化的测试数据。

数据集包含两类特殊样本，即功率一致性样本和零功率样本。功率一致性样本为尽可能使所有热源具有相同功率强度的样本。零功率样品表示部分热源为零功率强度的样品。在这里，我们构建了特殊的样本，其中分别有1/4,1/2,3/4，除1个热源外其他所有热源为零功率强度。6个测试集设置分别如下：

  • Test 0：所有样本的热源强度采用随机采样。

  • Test 1：所有样本的热源强度相同。

  • Test 2：1/4的热源为0功率热源，其余为功率随机采样热源。

  • Test 3：2/4的热源为0功率热源，其余为功率随机采样热源。

  • Test 4：3/4的热源为0功率热源，其余为功率随机采样热源。

  • Test 5：其中一个热源的功率采用随机采样，其余热源的功率为0。

评价准则

  • 平均绝对误差（MAE）。预测温度场绝对误差的平均值。
  • 绝对误差最大值（MaxAE）。预测温度场的最大绝对误差。
  • 组件区域平均绝对误差（CMAE）。热源组件上温度绝对误差的平均值。
  • 组件区域绝对误差最大值（MCAE）。热源组件上温度绝对误差的最大值。
  • 边界上平均绝对误差（BMAE）。边界上温度绝对误差的平均值。
